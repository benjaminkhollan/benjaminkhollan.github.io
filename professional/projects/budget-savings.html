Budget & Savings Calculator

Project Overview

The Budget & Savings Calculator is a web application that helps users manage their monthly income and expenses. It provides an intuitive interface for adding expenses, categorizing them, and calculating the remaining balance. Additionally, the savings calculator allows users to project their future savings based on initial investments, contributions, and interest rates.

Skills & Technologies Used

  • JavaScript
  • HTML5
  • CSS3
  • Flask (Backend)
  • Data Visualization
